# 🎯 Interest Matcher

## 📖 Project Overview

Interest Matcher is a web application that helps users discover and connect with interests that match their preferences. Users can select their interests from different categories, and the application suggests matching interests based on their choices.

This project was developed as part of an internship assignment to demonstrate frontend development skills using modern web technologies.

---

## 🚀 Features

- User-friendly and interactive interface
- Interest selection system
- Matching recommendations based on user choices
- Responsive design for desktop and mobile devices
- Clean and modern UI
- Fast performance using Vite

---

## 🛠️ Technologies Used

- React
- TypeScript
- Vite
- HTML5
- CSS3
- JavaScript
- Node.js

## ⚙️ Installation and Setup

### Step 1: Clone the Repository

```bash
git clone https://github.com/your-username/Decodelabs-week3.git
```

### Step 2: Navigate to the Project Folder

```bash
cd Decodelabs-week3
```

### Step 3: Install Dependencies

```bash
npm install
```

### Step 4: Start Development Server

```bash
npm run dev
```

### Step 5: Open in Browser

```text
http://localhost:5173
```

## 🎮 How to Use

1. Open the Interest Matcher application.
2. Browse available interest categories.
3. Select one or more interests.
4. Submit your selections.
5. View matching interests and recommendations.
6. Explore different combinations to discover new interests.
